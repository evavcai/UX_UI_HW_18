My UX Skills

User Research

User research is the most important step in the design process. I always begin a project with thorough interviews and data collection in order to identify the problem that needs solving.

User Interface Design

Once a solution is made, I begin to sketch, wireframe, and prototype the interface. The tools I use are Figma, Adobe, InVision, and Visual Studio Code.

Usability Testing

Iteration of the design only improves the outcome. I believe conducting in-depth usability tests is required to complete a project.
